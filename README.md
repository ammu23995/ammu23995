### 💼 Professional Experience

#### Software Engineer — Capital One *(formerly Discover Financial Services)*
📅 Jul 2023 – Present

- 🌐 Owned **FX-Gateway**, the settlement-critical service distributing real-time FX rates from NatWest to three payment networks (ACSS, NIS, SCIFI) across Discover Global Network. Sustained **99.95% availability** with strict p95 ingestion latency under 120ms across multi-network payment systems.
- 🤖 Built a real-time **decisioning engine** routing settlements into automated approval, manual review, or reject paths based on ML model score thresholds, backed by an H2O MOJO scoring service serving rate-risk predictions at p95 under 50ms. Raised straight-through processing and cut manual review volume **~20%**.
- 🛡️ Designed a **Blocking Audit Gate** validation layer across the multi-currency settlement pipeline, preventing partial-batch propagation at every stage. Settlement error propagation dropped **35%** (Sev-2 incidents per 1,000 file runs, 6-month trailing average).
- 💰 Owned the **Treasury Automation Processor**, a Lambda-triggered Spark job converting USD to acquirer settlement currencies across multi-region environments. Maintained a **99.94% SLO** over 12 months against a 4-hour settlement window; reduced manual reconciliation effort **~40%**.
- ⬆️ Migrated FX-Gateway from Pivotal Cloud Foundry to **Red Hat OpenShift** and upgraded Java 8 → 17 with JVM tuning (heap sizing, GC strategy, container-aware memory profiling). Deployment pipeline time reduced **30%**; infrastructure cost reduced **~25%** via right-sized container resources.
- 🔄 Led a **Jakarta EE migration** across the FX codebase, replacing deprecated javax dependencies and migrating 15+ Dozer mappers to compile-time MapStruct behind a feature flag with parallel diff-testing. Zero production downtime; full NatWest integration coverage preserved; delivered in under 4 weeks.
- 📊 Led the observability migration to **Datadog**, building SLO burn-rate dashboards, defining p95/p99 SLOs, and configuring end-to-end distributed tracing across all settlement services, with CloudWatch alarms auto-generating JIRA incidents and XMatters escalation. Critical incidents reduced **30%**; MTTR reduced **40%**.
